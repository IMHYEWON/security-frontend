# Node.js + Express를 사용한 HTTP 서버 구축하기



### 1. Node.js, npm 설치
``` bash
 ~ % brew install node    

 ~ % node -v
v22.0.0
  ~ % npm -v
10.5.1
```

### 2. package.json 생성 & Express 설치
``` bash
npm init -y
npm install express -save
```

### 3. 서버 구축
``` bash
node server.js
npm start
```
